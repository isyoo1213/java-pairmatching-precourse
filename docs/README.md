### Application 수행 절차에 따른 기능 흐름

1. 기능 선택 메시지 출력
2. 기능별 절차 수행
   1. 페어 매칭
      1. 과정/레벨/미션 출력
      2. 과정/레벨/미션 입력
      3. 기존 페어 매칭 이력 확인
         1. 존재하는 경우 - 재매칭 여부 출력/입력 수행
      4. 페어 매칭 수행
         1. 크루 이름 List<String> 형태로 준비
         2. 크루 이름 리스트 섞기
         3. 크루 2명씩 섞어서 매칭하기
            1. 홀수일 경우 - 마지막 페어에 마지막 크루를 매칭하기
         4. 크루 매칭 결과 점검
            1. 다른 레벨에서의 매칭 이력 확인
            2. 매칭 경우의 수 확인하기
            3. 3회 이상 매칭에 실패할 경우 - 오류메시지 출력
      5. 페어 매칭 결과 출력
   2. 페어 조회
      1. 과정/레벨/미션 출력
      2. 과정/레벨/미션 입력
      3. 기존 페어 매칭 이력 확인
         1. 존재하지 않는 경우 - 오류메시지 출력
      4. 페어 매칭 이력 출력
   3. 페어 매칭 초기화
      1. 모든 페어 매칭 결과 초기화
